# PMQ
PMQ (Python, Mosiac Qlabs) is a python program to be used as the interface for [The Art Firm's](www.theartfirmla.com) UFO build. This program acts as a GUI for users to have real time control over the connected Mosiac controller and Qlabs receiver. It has control over onboard: sound, lighting, mister system, and fogger system. 

# Installation
An executable is provided within /dist that can be downloaded and ran for the latest version.
